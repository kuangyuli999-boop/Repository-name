# Hermes Agent 自动化工作流 & GEO 内容分发优化系统

## 项目简介
以 Hermes Agent 为核心构建多智能体协同自动化工作流，配套搭建 GEO 全域内容分发与优化体系，通过定时任务实现 7×24 小时持续运行与自我迭代，打造「生成—评估—分发—反馈」全链路业务闭环。

## 核心架构
本项目采用多 Agent 分工协作模式，支持长链路推理与多轮自反馈：
- **Planner 智能体**：负责目标拆解、任务路径规划与执行策略制定
- **Executor 智能体**：承担内容生成、多平台适配与落地执行
- **Reviewer 智能体**：负责质量评估、效果复盘与生成策略修正

可全自动完成从选题策划、内容创作、结构优化到多平台发布的完整流程。

## 系统能力
- 深度适配各平台推荐算法，围绕高曝光入口定制内容版本
- 支持 A/B 测试与数据回流，反向迭代优化 Prompt 与生成策略
- 已实现稳定运行，单日可自动生成并分发数百条内容，生产效率提升 5 倍以上

## 工具与依赖
- 核心调度：Hermes Agent
- 开发与调试：Claude Code
- 底层模型：DeepSeek 系列

## 项目价值
验证了多智能体协同在规模化内容运营场景下的落地价值，通过自动化闭环实现内容生产效率与分发效果的双重提升。
